Optional Chaining

*The optional chaining operator (?.) allows you to access the value of a property located deep within a chain of objects without explicitly checking if each reference in the chain is null or undefined.
*if one of the reference in the chain is null or undefined,the optional chaining operator(?.)will sort circuit and return undefined.

Example

Program:
const user={

dog:{

name:"Alex"

}

};
console.log(user.cat?.name);//undefined

console.log(user.dog?.name);//Alex

console.log(user.cat.name);
Output:
Alex
