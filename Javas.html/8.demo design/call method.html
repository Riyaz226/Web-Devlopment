Call Method

*The Function type has a method named call() with the following syntax:
functionName.call(thisArg, arg1, arg2,....)
Syntax Information:
*The Call() method calls a function functionName with a given this value and arguments.
*The first arguments of the call() method thisArg is the this value.it allows you to set the this value to any given object.
*The remaining arguments of the call() method arg1,arg2.... are the arguments of the function.
*When you invoke a function,the JavaScript engine invokes the call() method of that function object.

Example

Program:
function add(a,b) {

return a+b ;

}
let result = add.call(this, 10, 20);

console.log(result)
Output:
30
Result:
In this example,instead of calling the add() function directly,we use the call() method to invoke the add() function.The this value is set to global object.
