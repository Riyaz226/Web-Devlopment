*The following Syntax:
(
function hello(){

console.log('hello')

}

)();
Syntax Information:
*A JavaScript immediately invoked function expersion is a function defined as an expersion and executed immediately after creation.
*The following shows the syntax of defining an immediately invoked function Expressions.

Example

Program:
let add =(function(a,b){

return a+b;})(10,20);
console.log(add);
Output:
30
Result:
In this example,instead of calling the add() function directly,we use the call() method to invoke the add() function.The this value is set to global object.
