Rest Operator

Rest Operator:
*ES6 provides a new kind of parameter so-called rest parameter that has a prefix of three dots(...).The rest parameter allows you to represent an indefinite number of arguments as an array.See the following syntax:
function fn(a,b,... args){

//...

}

Example

Program:
function sum(...args) {

let total =0;

for (const a of args) {

total +=a;

}

return total;

}
sum(1,2,3);
Output:
6
Result:
*args in an array,therefore,we could use the for of loop to iterate over its elements and sum them up. Assuming that the caller of the sum() function may pass arguments with various kind of data types such as number,string,and boolean and you want to calculater the total of number.
