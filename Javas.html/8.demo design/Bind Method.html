Bind Method

*The bind() method returns a new function,when invoked,has its this sets to a specific value.
*The following illustrates the syntax of the bind() method.
fn.bind(thisArg[, arg1[, arg2[, ...]]])
Syntax Information:
*In this syntax, the bind() method returns a copy of the function fn with the specfic this value (this Arg) and arguments(arg1,arg2,....).
*Unlike the call() and apply() methods,the bind() method doesn't immediately execute the function.It just returns a new version of the function whose this sets to thisArg arugument.

Example

Program:
let person = {

name : 'John Doe',

getName : function() {

console.log(this.name);

}

};
setTimeout(person.getName,1000);
Output:
undefined
=>When you pass a method an object is to another function as a callback,the this is lost.
Result:
*As you can see clearly from the output,the person.getName() returns undefined instead of 'John Doe'.
*This is because setTimeout() recived the function person.Name separately from the person object.
